GIT y GITHUB

En esta sección del curso usted podrá encontrar la introducción del Diseño Web así como los conceptos básicos de este así como el uso de estilos, implantación de contenido multimedia y el desarrollo de (Web Site) y otros temas de interés afines con esta temática así como el uso de GIT y GITHUB. En la actualidad la Web está presente en la vida cotidiana, cualquier individuo en especial los estudiantes de Diseño de aplicaciones o de sitios Web, saben o han escuchado mencionar sobre que se trata un sitio web. En este ámbito podemos afirmar que un sitio web es un conjunto de páginas web que se encuentran agrupadas bajo un dominio y las cuales comparten una dirección en la Web. En la actualidad se pueden encontrar en Interne muchos sitios Web o también conocidos como (Web Site) Es de resaltar que en el presente cualquier tipo de institución bien sea del sector público o privado posee un sitio en la Web acuñado por un dominio.
