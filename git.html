SISTEMA DE CONTROL DE VERSIONES GIT Y GITHUB

En la actualidad la Web está presente en la vida cotidiana, cualquier individuo en especial los estudiantes de Diseño de aplicaciones o de sitios Web, saben o han escuchado mencionar sobre que se trata un sitio web, GITHUB es una plataforma de desarrollo colaborativo de software para alojar proyectos usando el sistema de control de versiones Git. Github nos permite alojar nuestro repositorio de codigo y nos brinda herramientas utiles para el trabajo colaborativo

En este ámbito podemos afirmar que un sitio web es un conjunto de páginas web que se encuentran agrupadas bajo un dominio y las cuales comparten una dirección en la Web. En la actualidad se pueden encontrar en Interne muchos sitios Web o también conocidos como (Web Site) Es de resaltar que en el presente cualquier tipo de institución bien sea del sector público o privado posee un sitio en la Web acuñado por un dominio.

Instalacion del cliente github para escritorio

Para instalar github lo primero que debemos hacer es descargar su instalador de la pagina oficial, a la cual hemos dejado un link en la seccion de descargas de la pagina inicio, alli tendremos la opcion de descargar el instalador tanto para Windows como para Mac OS X, en nuestro caso seleccionamos la opcion para Windows y damos al boton descargar.

GITHUB es una plataforma de desarrollo colaborativo de software para alojar proyectos usando el sistema de control de versiones Git. Github nos permite alojar nuestro repositorio de codigo y nos brinda herramientas utiles para el trabajo colaborativo.

Para empezar a trabajar nuestros proyectos colaborativos lo primero que debemos hacer es crear una cuenta en github.com, una ves creada la cuenta en github se debera crear el repositorio dando clic en la opcion donde dice nuevo repositorio.

Luego deberemos asignar un nombre a nuestro repositorio, este nombre no puede contener espacios ya que este es el mismo que tendra nuestra pagina si la publicamos en github pages, tambien tenemos un espacio para colocar una breve descripcion del contenido de nuestro repositorio. Tambien se puede indicar si el repositorio sera publico o no, por el cual solo sera accesible para el usuario que lo crea y los contribuidores que este asigne, pero debemos tener en cuenta que para poder crear un repositorio privado tendremos que actualizar nuestra cuenta por 7 dolares al mes.
