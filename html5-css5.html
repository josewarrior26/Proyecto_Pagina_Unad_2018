HTML5 o como su nombre lo indica (HyperText Markup Language, versión 5) es la quinta versión del lenguaje HTML. Esta herramienta en asociación con CSS3, redefine los nuevos y novedosos estándares de desarrollo de sitios web (Web Site), se ha rediseñando el código fuente logrando dar solución a problemas hallados, actualizándolo para estar acorde a las posibles nuevas necesidades. Este no se limita únicamente a crear nuevas etiquetas o atributos, sino que además a esto incluye nuevas características proporcionando una novedosa plataforma de desarrollo de aplicaciones web. (Arkaitz (2014), https://www.arkaitzgarro.com/html5/capitulo-1.html)

Teniendo en cuenta que HTML permite al diseñador definir la estructura de una página web, entonces las hojas de estilo en cascada o más conocidas en el ámbito del diseño web como (Cascading Style Sheets o CSS) son las que nos brindan la posibilidad de definir ciertos estilos además de reglas de representación para los diferentes dispositivos que se encuentran en el mercado común , bien sea sean pantallas de equipos de escritorio, impresoras o portátiles así como en los dispositivos móviles (Smartphone), u otros dispositivos con la capacidad de mostrar el diferente contenidos web, con las hojas de estilo se puede definir de manera eficaz la presentación de las páginas creadas por el diseñador y por ende debe ser uno de los conocimientos esenciales que cualquier diseñador web debe manipular de forma completa y eficiente y así lograr realizar el trabajo a cargo a toda cabalidad.
