Bienvenidos al curso Diseño de Sitios Web

 

El curso está dirigido a estudiantes que se encuentren interesados en el diseño de sitios web y la administración elemental de estos sitios. El curso en general muestra la introducción, profundización de temáticas asociadas a este tema además de presentar información suplementaria sobre el uso del cliente GIT y GIHUB para gestionar proyectos colaborativos de diseño de sitios web, así como HTML5 y CSS3 los cuales son utilizados para la creación de contenidos en formato web como sitios Web páginas.

Estructuración del curso

 

Unidad 1. Introducción al Diseño Web. Unidad 2. HTML 5. Unidad 3. CSS3. Las tres unidades que constituyen este curso abordaran temas de desarrollo, ejecución y administración de proyectos de diseño web, además a esto permite el uso de las Tics como agente intermediario entre el curso y el Estudiantes; para esto la página cuenta con elementos de aprendizaje interactivo como OVAS, Tutoriales y Vídeos además de lecturas, que pueden ser complementadas con actividades académicas en línea y el manejo de canales sincrónicos y asíncronos de comunicación para el avance del proceso académico que propone el curso Diseño de sitios web
